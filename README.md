# SPICE - Street Food Platform

## Project Description

SPICE (Street Food Platform) is a Database Management System (DBMS) project developed as part of the IT214 Database Management System course.

The system manages street food vendors, stalls, food items, menus, customer reviews, hygiene inspections, licenses, and authorities. It also calculates a Trust Score to help customers identify reliable vendors based on hygiene ratings, customer reviews, inspection frequency, and vendor experience.

---

## Features

- Vendor Management
- Stall Management
- Food Category Management
- Food Item Management
- Menu Management
- Customer Reviews and Ratings
- Hygiene Inspection Tracking
- License Management
- Trust Score Calculation
- SQL-based Data Analysis
